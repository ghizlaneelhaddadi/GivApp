#  Application mobile de donation humanitaire

Cette application mobile Android vise à faciliter les **dons humanitaires** via une interface simple, moderne et accessible.  
Elle est développée **en Kotlin** et utilise **Firebase** pour la gestion des utilisateurs, la base de données, le stockage et les notifications en temps réel.

## Technologies utilisées

- **Kotlin (Android)**  
- **Firebase Authentication**  
- **Firebase Firestore**  
- **Firebase Storage**  
- **Firebase Cloud Messaging (FCM)**  
- **Material Design UI**
